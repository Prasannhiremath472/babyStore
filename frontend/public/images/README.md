## Recommended Image Sizes

| Folder     | Size         | Format |
|------------|--------------|--------|
| hero/      | 1200 x 600px | JPG    |
| categories/| 500 x 400px  | JPG    |
| products/  | 500 x 500px  | JPG    |
| banners/   | 700 x 350px  | JPG    |
| store/     | 200 x 150px  | JPG    |

## Tips
- Keep file sizes under 200KB for fast loading
- Use JPG for photos, PNG for logos/icons
- All images are served with browser caching
